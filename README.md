# Bouldershare-app

Bouldershare 是一款面向抱石（bouldering）爱好者的移动应用，用户可以在无需登入的情况下快速上传、浏览和分享抱石路线。

本项目旨在为社区提供一个轻量、易用的平台，帮助大家学习和交流登山技巧。

## 功能特性

- 🧗 无需注册/登入即可上传路线
- 📷 支持图片和描述添加详尽的路线信息
- 🔎 浏览并搜索其他用户上传的路线
- ⭐ 收藏/点赞感兴趣的路线
- 📍 路线位置标记

## 运行与开发指南

1. 克隆仓库：
   ```bash
   git clone https://github.com/school98072/Bouldershare-app.git
   cd Bouldershare-app
   ```
2. 安装依赖（根据具体技术栈，例如 Node、Flutter 等）：
   ```bash
   # 示例：
   npm install
   # 或者
   flutter pub get
   ```
3. 启动开发服务器或模拟器：
   ```bash
   npm start
   # 或者
   flutter run
   ```
